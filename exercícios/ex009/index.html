Para criar uma ordered list utilizamos a tag <ol>, como o nome indica são listas ordenadas e utilizamos a tag <li> (list item) para identificar cada item da lista.

Caso tenhamos de adicionar um item no princípio da lista, podemos adicionar e a própria tag <ol> irá ordenar correta e automaticamente a numeração dos itens.

Dica: Caso queiramos utilizar apenas <li> sem o fechamento da tag podemos fazê-lo, apartir da evolução do HTML5 é opcional a forma de usar a tag.

Nota: A tag <ol> possui um parâmetro type onde configuramos o tipo de marcador da lista criada.

As opções de valores para esse parâmetro são:
1 - Valor padrão. Cria listas numeradas. EX: 1, 2, 3, 4, 5, ...
A - Cria listas alfabéticas em maiúsculas. EX: A, B, C, D, E, ...
a - Cria listas alfabéticas em minúsculas. EX: a, b, c, d, e, ...
I - Cria listas com algarismos romanos em maiúsculas. EX: I, II, III, IV, V, ...
i - Cria listas com algarismos romanos em minúsculas. EX: i, ii, iii, iv, v, ...

Listas Não Ordenadas - Unordered Lists

São listas em que a ordem dos items não influenciará no significado da mesma.


Para criar uma unordered list utilizamos a tag <ul>, como o nome indica são listas desordenadas e utilizamos a tag <li> (list item) para criar cada um dos itens internos.

O marcador padrão é a bolinha preta com preenchimento (disc) mas existe a opção de configurar a propriedade type da tag <ul> com os seguintes valores:

Disc - Marcação por padrão. Uma bola preta totalmente preenchida.
Circle - Uma bola com borda preta e sem preenchimento interno.
Square - Um pequeno quadrado preto totalmente preenchido.
