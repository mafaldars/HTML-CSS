Citações completas

Segundo Jeff Nobel, no seu livro de HTML para leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos neste capítulo descrevem os blocos de texto.
A tag <blockquote> é um bloco de citações para determinados autores. Verificamos o espaçamento para dentro (margem) na tela como sinal. Dentro desta tag temos o parâmetro cite que indica a citação advém de determinada fonte, um livro neste caso.
Visualmente o cite com o link do livro não irá aprecer mas os mecanismos de busca do site e do Google (parte interna do site) saberão que a citação pertence a este livro.
