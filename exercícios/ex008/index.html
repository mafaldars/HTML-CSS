Formatações principais

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em itálico usando a tag <i> (não semântico).

Outra dica: se selecionarmos o texto que pretendemos colocar em itálico, clicamos Command + Shift + P abre barra de pesquisa (acima) do VSCode e clicamos I + ENTER , o texto ficará automaticamente inserido na tag <i>.

Nesta frase temos um termo com ênfase usando a tag <em> (semântica).

Outra dica: se selecionarmos o <p> texto que pretendemos colocar dentro de um parágrafo <p>, clicamos Command + Shift + P abre barra de pesquisa (acima) do VSCode e clicamos P + ENTER, o texto ficará automaticamente inserido na tag <p>.

Texto demarcado

Nesta frase temos um termo valioso/demarcado usando a tag <mark>. Demarcar com esta tag é como se estivessemos a demarcar com marcador de uma cor vibrante para realçar a importância do conteúdo.

Neste parágrafo temos outro termo demarcado no final. Não herda a cor do parágrafo anterior (fizemos CSS inline no anterior). Para herdar a mesma cor em ambos os parágrafos basta retirar o CSS inline do parágrafo anterior (temos style no head da página para ambas as tags mark.)

Texto grande e pequeno

Estamos a criar um texto grande e um texto pequeno neste parágrafo.
A tag <big> está obsoleta apesar de ainda funcionar.
A tag <small> está operacional (ex: as letras minúsculas dos contratos devem ser lidas devido à sua importância apesar de serem em tamanho pequeno.)

Texto delete

Podemos marcar um texto como excluído para indicar que deve ser lido mas não deve ser considerado.
A tag <del> é utilizada para este efeito (semântica).

Texto inserido

Podemos marcar um texto como inserido para dar ênfase (sublinhado) e indicar que foi adicionado.
A tag <ins> é utilizada para este efeito (semântica).

Existe também a tag <u> para sublinhar (não semântica). Poderá ficar obsoleta.

Texto sobrescrito

Para inserir coisas do tipo x20+3.
A tag <sup> é utilizada para este efeito (cima).

Texto subscrito

Para inserir coisas do tipo H20.
A tag <sub> é utilizada para este efeito (baixo).
