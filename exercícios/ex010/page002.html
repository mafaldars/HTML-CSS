Esta página é uma cortesia do Google.

Nota: o atributo rel="nofollow" significa que não damos consentimento, o valor "dofollow" significa que damos aval.

Existe um recurso interessante para links que é indicar qual é a natureza do destino usando o atributo rel. Este atributo aceita vários valores, entre eles:
next - indica que o link é para a próxima parte (página) do documento atual - ou avanço na página de navegação, não necessariamente a segunda página;
prev (previous) - indica que o link é para a parte (página) anterior do documento atual;
author - indica que é um link para o site do autor do artigo atual;
external - indica que é um link para outro site que não faz parte do site atual (o nosso, por exemplo).
