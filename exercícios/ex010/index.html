Utilizar links externos

Pode aceder ao meu repositório público no GitHub.

Nota: Quando acedemos através de um link externo o user é redirecionado diretamente para outro site, sobrepondo o nosso (comportamento não desejado). Para isso não acontecer devemos colocar os atributos target="_blank" rel="external".
Não colocamos target="_blank" rel="external" quando se trata de um hyperlink para o nosso próprio site.

Resumindo, para podermos controlar onde o site de destino irá abrir, utilizamos o atributo target que suporta os seguintes valores:
_blank - irá abrir o link numa nova janela (tab) em branco;
_self - vai abrir o link na janela (tab) atual (padrão);
_top - vai desfazer todos os frames e abrir o destino no navegador completo;
_parent - similar ao uso do _top em uma referência à janela mãe nome-do-frame caso esteja usando frames, indicar o nome da janela a abrir.
Nota: Foquemo-nos apenas em _blank e _self pois o uso de frames é uma técnica quase em desuso.

Os hyperlinks são um dos conceitos mais antigos da história da linguagem HTML. Permitem que liguemos um ponto a outro da Wordl Wide Web (sub-rede da Internet).

Cada vez que acedemos a um site e clicamos para sermos direcionados para outra page, outro site ou até para fazer download de um arquivo, estamos a interagir com um hyperlink.

Os mecanismos de busca utilizam-se de hyperlinks de um site.

O Google, por exemplo, para encontrar um determinado site, vasculha constantemente todos os outros sites da Internet em busca de outros links para descobrir novos conteúdos. Por isso é que é tão importante conseguir links válidos de outros sites para o nosso próprio site.

Nota: Podemos ver detalhadamente como funcionam os algoritmos fundamentais de busca com o vídeo de um engenheiro da Google, o Matt Cutts.

Para criar um hyperlink devemos criar âncoras através da tag <a>. O principal atributo dessa tag é o href, onde iremos colocar a URL completa para outro site.

EX: <a href="https://gustavoguanabara.github.io">Aceda ao perfil GitHub</a>.

Outro atributo útil da tag é o hreflang, que permite indicar qual é o idioma principal do site para onde o link irá desviar o fluxo de navegação. Isto permite informar o browser e o software de tradução como lidar caso o user opte por traduzir automaticamente os conteúdos.

EX: <a href="https://www.w3schools.com/html/" hreflang="en"> Site da W3Schools em Inglês</a>
