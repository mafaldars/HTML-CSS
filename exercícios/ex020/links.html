Os pseudo-elementos (em CSS = ::) são palavras-chave adicionadas às declarações de um seletor e permitem formatar um pedaço específico de um elemento. Pseudo-elementos podem ser utilizados em class, id, nos elementos e mexem no conteúdo periférico do elemento.
Os principais pseudo-elementos usados nas CSS são ::before, ::after, ::first-letter, ::first-line.
Os pseudo-elementos ::after e ::before indicam que podemos mostrar coisas antes e depois do elemento.

Seletores personalizados:

  • # = id
  • . = class
  • : = pseudo-class
  • :: = pseudo-element
  • > = children

Nota: O uso das pseudo-classes e dos pseudo-elementos é para personalização.
