Uma <div> ocupa a largura da tela caso não tenhamos declarado um valor para width.

Pseudo-classes (em CSS = :) são palavras-chave adicionadas às declarações de um seletor e especificam um determinado estado de um elemento dentro do HTML, ou seja, conseguimos configurar o estado.

Existem várias pseudo-classes para estilos, tais como:
  • :hover
  • :visited
  • :active
  • :checked
  • :empty
  • :focus

Hover é uma pseudo-classe de vários elementos, significa que quando eu passar o mouse por cima teremos um efeito.
