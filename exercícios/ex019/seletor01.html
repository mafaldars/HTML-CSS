Nota: Num documento HTML só podemos utilizar um id (em CSS = #) para um único elemento, o que significa que não podemos ter dois elementos com o mesmo id dentro da mesma página. Porém, podemos atribuir class (em CSS = .) para vários elementos que possuam a mesma característica.
Conseguimos adicionar mais do que uma class ao mesmo elemento (usar barra de espaço para separar as class) e conseguimos usar class num id, lembrando que as declarações dadas no id sobrepõem-se sempre às class.
