Olá, Mundo!

O dafont.com é um site com várias fontes não suportadas no Google Fonts.

Tipos de format()
- opentype (otf)
- truetype (ttf)
- embedded-opentype
- truetype-aat ( Apple Advanced Typography)
- svg
Nota: Devemos pesquisar que tipo de formato suporta cada browser e que tipo de format() é cada font.
Caso tenhamos as CSS numa folha externa no path temos de colocar ../fontes/OldLondon.otf,etc.
