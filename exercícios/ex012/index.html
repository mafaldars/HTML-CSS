Inserir vídeos com hospedagem local

Este vídeo encontra-se hospedado no meu próprio servidor.

Nota: Safari não funciona com o formato OGV.

Como fizemos anteriormente com os áudios, o processo será identico para os vídeos com as tags para vídeo.
Veja o exemplo a seguir:



Dentro da tag <video> iremos inserir
<source src="midia/meu-video.mp4" type="video/mp4">
<source src="midia/meu-video.m4v" type="video/m4v">
<source src="midia/meu-video.webm" type="video/webm">
<source src="midia/meu-video.ogv" type="video/ogv">
</video>

Nota: Relembramos de que as instruções são lidas de cima para baixo, logo o browser irá tentar reproduzir primeiro o vídeo mp4 (primeiro), depois o vídeo em formato m4v e assim sucessivamente.
Caso o browser não consiga reproduzir nenhum dos formatos inseridos, irá apresentar uma mensagem ao user a informar e o link para efetuar o download.

EX do código e da mensagem:<p><a href="midia/meu-video.mp4" download="meu-video.mp4" type="application/video">Clique aqui para efetuar o download do arquivo MP4.</a></p>

Clique aqui para efetuar o download do arquivo MP4.

Nota: Temos de ter em atenção o tamanho dos vídeos pois se formos multiplicar o tamanho dos vídeos pelas possíveis visualizações, estamos a falar em muitos MB e até GB, isso custaá muito dinheiro de hospedagem do nosso servidor.

Inserir vídeos do YouTube

Nota: Colocar vídeos do YouTube (ou do Vimeo) mesmo que os users visualiazem vezes sem conta, não consome dados do nosso servidor, consome do servidor do YouTube.
Caso o cliente pretenda mesmo ter os vídeos hospedados no próprio servidor, devemos informar de que podemos fazê-lo mas que isso terá custos elevados a cargo do cliente.
